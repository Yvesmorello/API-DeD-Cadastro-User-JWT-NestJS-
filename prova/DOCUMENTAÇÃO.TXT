COMO USAR A API

INICIE O PROGRAMA COM O COMANDO NPM RUN START 
ABRA O INSOMNIA OU OUTRO PROGRAMA DE ROTAS 


--- USER ---

-PRIMEIRAMENTE VOCE DEVE CRIAR UM USUARIO NA ROTA (http://localhost:3000/user), INFORMANDO USERNAME E PASSWORD
-EM SEGUIDA NA ROTA LOGIN USER (http://localhost:3000/login), INFORME SEU USERNAME E PASSWORD PARA LOGAR NO SISTEMA
-APÓS LOGAR SERA GERADO UM TOKEN, QUE É OBRIGATÓRIO PARA UTILIZAR AS OUTRAS ROTAS

-PARA UTILIZAR AS OUTRAS ROTAS DA API VOCE DEVE INSERIR O TOKEN GERADO NO LOGIN NA ABA (AUTH- SELECIONE A OPÇÃO BEARER TOKEN) E INSIRA O TOKEN NO LOCAL(TOKEN). COM ISSO O USO DA ROTA SERA LIBERADO 

-VOCE TEM A POSSIBILIDADE DE:
-BUSCAR OS USUARIOS NA ROTA GET(POR ID{http://localhost:3000/user/:ID}, POR NOME{http://localhost:3000/user/username/{USERNAME}} E GERAL{http://localhost:3000/user}),
-ATUALIZAR SEU USUARIO NA ROTA PUT{http://localhost:3000/user/:ID} 
-EXCLUIR SEU USUARIO NA ROTA DELETE{http://localhost:3000/user/:ID}
-SEMPRE UTILIZANDO O TOKEN DE AUTORIZAÇÃO


--- PERSONAGEM ---

-PARA UTILIZAR AS ROTAS DO PERSONAGEM, INSIRA O TOKEN NO LUGAR ESPECIFICADO ANTERIORMENTE EM CADA ROTA QUE FOR UTILIZAR 
-PARA CRIAR SEU PERSONAGEM VA NA ROTA POST CHARACTER (http://localhost:3000/character) E INSERIR AS INFORMAÇÕES DO SEU PERSONAGEM
-AS INFORMAÇÕES DISPONIVEIS SÃO (NOME, LEVEL, ABILITY_SCORE, CLASSE, FEAT, ALIGNMENT, SPELL E ITEM)
-CASO VOCE INSIRA UMA INFOMRAÇÃO QUE NAO CONDIZ COM OS DADOS DO D&D, A API RETORNARA UM ERRO, MOSTRANDO AS POSSIVEIS INFORMAÇÕES QUE PODEM SER COLOCADAS NO ATRIBUTO ESPECIFICO
-É POSSIVEL:
-LISTAR PERSONAGENS NA ROTA GET(http://localhost:3000/character)
-ATUALIZAR PERSONAGENS NA ROTA PUT(http://localhost:3000/character/:ID) - TAMBEM SERA VALIDADO OS DADOS CONFORME A API 
-EXCLUIR NA ROTA DELETE(http://localhost:3000/character/:ID)
-SEMPRE COM O TOKEN DE AUTORIZAÇÃO

-------------

AO CRIAR SEU USUARIO E SEU PERSONAGEM, ELES SERÃO SALVOS NO BANCO DE DADOS, ASSIM COMO OS LOGS DE TODAS AS REQUISIÇÕES FEITA PARA A API
